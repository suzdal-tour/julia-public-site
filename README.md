# Run locally


  - Clone this repository and its submodules
  - Set `ACT_AUTH` environment variable
  - Set `ACT_DATA` environment variable
  - Run the site with the following command: `hugo serve --gc --tlsAuto`
  - Run the CF according to their instructions in their repositories
